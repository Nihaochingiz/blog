# blog
Приложение blog на fastAPI

## Запуск сервера

Из директории `blog/src`:
```bash
uvicorn blog.main:app --reload
```

Сервер запускается на `http://127.0.0.1:8000`

## Эндпоинты

### GET /articles
Получение списка всех статей

**Ответ:**
```json
{
  "items": [
    {
      "id": "uuid",
      "title": "Заголовок",
      "content": "Содержание"
    }
  ]
}
```

### POST /articles
Создание новой статьи (требует авторизацию администратора)

**Тело запроса:**
```json
{
  "title": "Заголовок",
  "content": "Содержание"
}
```

**Параметры авторизации (в теле запроса):**
```json
{
  "username": "admin",
  "password": "secret"
}
```

**Ответ (201 Created):**
```json
{
  "id": "uuid",
  "title": "Заголовок",
  "content": "Содержание"
}
```

**Ошибки:**
- 401 Unauthorized - неверные учетные данные
- 403 Forbidden - недостаточно прав (требуется администратор)
